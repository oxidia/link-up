# Video Call Application

## Name: Link Up

## Key features

- Specify username
- List calls history
- Join a Video/Audio call
- Enable/Disable Microphone
- Enable/Disable Camera

## Architecture

- Clean Architecture

## Stack

- kotlin
- Jetpack compose
- Stream(https://getstream.io)
- Dagger Hilt
- Room database
- Coroutines
